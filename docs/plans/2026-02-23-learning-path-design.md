# 学習パス（ロードマップ）機能 設計書

**目的:** 各言語の既存 overview データ（特性・概念・文法）から学習順序を自動導出し、ロードマップ形式で表示する。初心者と経験者の2ルートを提供。

**対象言語:** JavaScript, Java, jQuery（既存の overview データを持つ全言語）

---

## 1. 学習パス自動生成ロジック

### ノード分類（3層）

| レベル | データソース | 意味 |
|---|---|---|
| 基礎 | `overview.characteristics` | 言語特性（根幹） |
| 中級 | `overview.concepts` | 概念（特性に紐づく） |
| 実践 | `versions[].terms` | 文法・API（概念に紐づく） |

### 自動導出ルール

1. 特性を根ノード（ブランチの起点）として配置
2. 概念を `characteristicId` で特性の下に配置
3. 文法を `relatedTermIds` で概念の下に配置
4. 各ノードのレベルはデータソースから自動判定
5. 文法が複数概念に関連する場合は、最初の概念のみに配置

### ルート切替

- **初心者ルート**: 全ノード展開表示。基礎→中級→実践の流れを強調
- **経験者ルート**: 基礎（特性）を折りたたみ表示。概念→文法を中心にハイライト

## 2. UI デザイン

### タブ構成（3タブ）

```
┌──────────────┐ ┌──────────────┐ ┌──────────────┐
│   言語概要   │ │   学習パス   │ │ バージョン履歴 │
└──────────────┘ └──────────────┘ └──────────────┘
```

### ロードマップ表示

各特性がブランチの起点。ツリー構造で概念→文法を展開:

```
📘 動的型付け ─────────────── 基礎
│
├── 📗 型変換         ┌─ 📗 ホイスティング
├── 📗 typeof 演算子  │
│                     └── 📙 let / const
└── 📗 スコープ ──────────┘

📘 第一級関数 ─────────────── 基礎
│
├── 📗 クロージャ ──── 📙 Arrow Functions
├── 📗 コールバック ── 📙 Promise
└── 📗 高階関数
```

### コンポーネント構成

| コンポーネント | 状態 | 役割 |
|---|---|---|
| `TabNav.vue` | 変更 | 「学習パス」タブを追加 |
| `App.vue` | 変更 | LearningPathTab の統合 |
| `LearningPathTab.vue` | 新規 | ロードマップ生成・ルート切替・表示 |
| `RoadmapBranch.vue` | 新規 | 1つの特性ブランチ |
| `RoadmapNode.vue` | 新規 | 個別ノード（レベル色分け・クリックジャンプ） |

### ノードクリック動作

- 基礎/中級ノード → 「言語概要」タブの該当カードにジャンプ
- 実践ノード → 「バージョン履歴」タブの該当 TermCard にジャンプ
- 既存の `jumpToTerm` 機構を再利用
